## Prerequisites

- **Deno 2.0+** ([install here](https://deno.land/manual/getting_started/installation))
- **API Keys** - Get an Anthropic API key ([here](https://console.anthropic.com)) and Firecrawl API key ([here](https://firecrawl.dev))
  - *Note: This example uses Anthropic's Claude, but you can use OpenAI or other providers (see [LLM Providers](/zypher/core-concepts/llm-providers))*
  - *Firecrawl is just an example MCP server for web crawling - you can use any MCP server or none at all*

## Step 1: Install Zypher Agent

First, initialize a new Deno project (if you haven't already):

```bash
deno init
```

Then add Zypher Agent to your project:

```bash
deno add jsr:@zypher/agent
deno add npm:rxjs-for-await
```

## Step 2: Set Up Environment Variables

Create a `.env` file in your project root:

```bash
ANTHROPIC_API_KEY=your_anthropic_api_key_here
FIRECRAWL_API_KEY=your_firecrawl_api_key_here
```

## Step 3: Create Your First Agent

Create a new file called `main.ts`:

*This example demonstrates the core pattern: create an agent with your preferred LLM provider, optionally register MCP servers for external capabilities, then run tasks. We're using Firecrawl as an example MCP server to enable web crawling.*

```typescript
import { createZypherAgent } from "@zypher/agent";
import { createFileSystemTools } from "@zypher/agent/tools";
import { eachValueFrom } from "rxjs-for-await";

// Create the agent with the simplified factory function
const agent = await createZypherAgent({
  model: "claude-sonnet-4-5-20250929",
  tools: [...createFileSystemTools()],
  mcpServers: [
    {
      id: "firecrawl",
      type: "command",
      command: {
        command: "npx",
        args: ["-y", "firecrawl-mcp"],
        env: {
          FIRECRAWL_API_KEY: Deno.env.get("FIRECRAWL_API_KEY")!,
        },
      },
    },
  ],
});

// Run a task - the agent will use web crawling to find current AI news
const event$ = agent.runTask("Find latest AI news");

// Stream the results in real-time
for await (const event of eachValueFrom(event$)) {
  if (event.type === "text") {
    Deno.stdout.writeSync(new TextEncoder().encode(event.content));
  }
}
```

<Callout type="info" title="Alternative: Manual Setup">
For more control, you can also create agents manually:

```typescript
import {
  AnthropicModelProvider,
  createZypherContext,
  ZypherAgent,
} from "@zypher/agent";

const zypherContext = await createZypherContext(Deno.cwd());
const agent = new ZypherAgent(
  zypherContext,
  new AnthropicModelProvider({
    apiKey: Deno.env.get("ANTHROPIC_API_KEY")!,
  }),
);
```
</Callout>

## Step 4: Run Your Agent

Execute your agent:

```bash
deno run -A main.ts
```

You'll see the AI agent respond with an explanation of what AI agents are.

## That's it! 

You've created your first Zypher Agent with just a few lines of code. The basic pattern is:

1. **Install** - `deno add jsr:@zypher/agent`
2. **Create** - Agent with your preferred LLM provider
3. **Run** - Any task with `agent.runTask()`

## Try Different Tasks

Change the task string to try different things:

```typescript
// Ask questions
agent.runTask("What are the benefits of using AI agents?")

// Give instructions
agent.runTask("Write a hello world program in TypeScript")

// Request analysis
agent.runTask("List the pros and cons of different programming languages")
```

## Interactive CLI Mode

For a more interactive experience, you can use the built-in terminal interface with `runAgentInTerminal`. This provides a chat-like interface where you can have conversations with your agent:

```typescript
import { runAgentInTerminal } from "@zypher/agent";

// const agent = new ZypherAgent(...)
// await agent.init()

// Start interactive CLI
await runAgentInTerminal(agent, "claude-sonnet-4-5-20250929");
```

This opens a terminal interface where you can:
- Chat with your agent interactively
- See real-time responses as they stream
- Ask follow-up questions in the same session
